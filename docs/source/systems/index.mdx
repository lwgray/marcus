## System Categories

Marcus is built on 50+ interconnected systems organized into categories:

### 🧠 Core Intelligence
Memory systems, AI engine, learning, task intelligence, recommendations

### 🤝 Agent Coordination
Agent management, worker support, context dependencies, communication hub

### 📊 Project Management
Project control, Kanban integration, analysis tools, reporting, workflows

### 🏗️ Infrastructure
Error framework, persistence layer, workspace isolation, NLP, configuration

### 🏛️ Architecture
MCP server, event-driven architecture, service registry

### ✅ Quality Systems
Monitoring, quality assurance, detection systems, testing, health analysis

### ⚙️ Operations
Cost tracking, pipelines, operational modes

### 💻 Development
Code analysis systems, API systems

### 📈 Visualization
Logging system, visualization tools

### 🔒 Security
Security systems, access control
