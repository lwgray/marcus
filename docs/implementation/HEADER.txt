## Overview

This plan addresses 5 critical MVP concerns:

1. **Packaging** - Centralized configuration (Issue #68)
2. **Telemetry** - User journey tracking
3. **Observability** - Feature-level status with git context
4. **Professional Git Workflows** - Workspace isolation with automatic branching
5. **Research Instrumentation** - MAS study enablement

**Key Principles**:
- Build incrementally (no big-bang rewrites)
- Maintain backward compatibility
- Test-driven development (TDD)
- Document as you go
